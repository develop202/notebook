# npm缓存

<mark>npm</mark>缓存目录

```shell
npm config get cache
```

清理<mark>npm</mark>缓存

```shell
npm cache clean --force
```

# yarn缓存

<mark>yarn</mark>缓存目录

```shell
yarn cache dir
```

清理<mark>yarn</mark>缓存

```shell
yarn cache clean
```

# pip缓存

<mark>pip</mark>缓存目录

```shell
pip cache info
```

清理<mark>pip</mark>缓存

```shell
pip cache purge
```

# go缓存

清除<mark>模块</mark>缓存

```shell
go clean -modcache
```
