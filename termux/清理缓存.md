# npm缓存

==npm==缓存目录

```shell
npm config get cache
```

清理==npm==缓存

```shell
npm cache clean --force
```

# yarn缓存

==yarn==缓存目录

```shell
yarn cache dir
```

清理==yarn==缓存

```shell
yarn cache clean
```

# pip缓存

==pip==缓存目录

```shell
pip cache info
```

清理==pip==缓存

```shell
pip cache purge
```

# go缓存

清除==模块==缓存
```shell
go clean -modcache
```
