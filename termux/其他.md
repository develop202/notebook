```shell
use-black-ui=true
```

# 编译kulala-fmt(.http格式化工具)

### 克隆远程库到本地

```shell
git clone https://github.com/mistweaverco/kulala-fmt
```

### Termux下修改文件

将==Makefile==文件大约==第6行Linux==的架构amd64改为==arm64==

### 创建build

```shell
mkdir build
```

### 进入build命令编译(需要go环境)

```shell
make builds
```
